# toxicity-detect
Toxicity Detection in Online Comments

This project analyzes toxic online comments using transformer-based NLP models.

Dataset:
- Toxic Comment Classification Challenge dataset from Hugging Face

Models:
- RogueBERT-Toxicity-85K
- toxic-bert

Evaluation Metrics:
- Accuracy
- Precision
- Recall
- F1-score

Libraries:
- pandas
- sklearn
- transformers
- matplotlib
- seaborn
